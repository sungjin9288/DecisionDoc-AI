# 개인정보처리방침

**DecisionDoc AI** (이하 "서비스")는 「개인정보 보호법」 및 관련 법령을 준수하며,
이용자의 개인정보를 소중히 보호합니다.

---

## 제1조 (개인정보 수집 항목 및 수집 방법)

### 수집 항목
| 구분 | 수집 항목 | 수집 목적 |
|------|-----------|-----------|
| 필수 | 아이디, 비밀번호(해시), 이메일 | 회원 가입 및 인증 |
| 선택 | 소속 기관, 부서명 | 맞춤형 서비스 제공 |
| 자동 수집 | IP 주소, 접속 시간, 브라우저 정보 | 보안 감사 및 서비스 개선 |
| SSO 연동 시 | 기관 계정 정보(LDAP/SAML 제공) | 인증 목적 (저장하지 않음) |

### 수집 방법
- 서비스 가입 시 이용자 직접 입력
- SSO 연동 시 기관 시스템에서 자동 수신

---

## 제2조 (개인정보 이용 목적)

수집된 개인정보는 다음 목적으로만 이용됩니다:

1. 회원 식별 및 서비스 이용 인증
2. 결재 워크플로우 참여자 식별
3. 알림 발송 (이메일/슬랙)
4. 보안 감사 및 이상 접근 탐지
5. 서비스 이용 통계 분석 (비식별화)

---

## 제3조 (개인정보 보유 및 이용 기간)

| 항목 | 보유 기간 |
|------|-----------|
| 회원 정보 | 탈퇴 시 즉시 삭제 |
| 감사 로그 | 1년 (법령 의무) |
| 접속 로그 | 90일 |
| 문서 데이터 | 프로젝트 보관 정책에 따름 |

---

## 제4조 (개인정보 제3자 제공)

원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다.
단, 다음의 경우는 예외입니다:

- 이용자의 사전 동의가 있는 경우
- 법령의 규정에 따라 수사기관의 요청이 있는 경우

### 외부 서비스 연동
| 서비스 | 목적 | 전송 데이터 |
|--------|------|-------------|
| OpenAI API | 문서 생성 | 문서 내용 (개인식별정보 제외 권고) |
| Stripe | 결제 처리 | 이메일, 카드 정보 (Stripe 서버에만 저장) |
| SMTP 서버 | 알림 발송 | 이메일 주소 |

---

## 제5조 (개인정보 처리 위탁)

서비스 제공을 위해 아래와 같이 개인정보 처리를 위탁합니다:

| 수탁자 | 위탁 업무 |
|--------|-----------|
| 클라우드 인프라 제공자 | 서버 운영 및 데이터 보관 |

---

## 제6조 (정보주체의 권리)

이용자는 언제든지 다음 권리를 행사할 수 있습니다:

- **열람**: 본인 개인정보 열람 요청
- **정정**: 부정확한 정보 수정 요청
- **삭제**: 개인정보 삭제 요청 (회원 탈퇴)
- **처리정지**: 개인정보 처리 정지 요청

요청 방법: 관리자 계정에서 직접 처리 또는 **개인정보 보호책임자**에게 연락

---

## 제7조 (개인정보 보호 기술적 조치)

| 조치 | 내용 |
|------|------|
| 암호화 | 비밀번호 bcrypt 해시, SSO 시크릿 AES-256 |
| 전송 보안 | TLS 1.2 이상 |
| 접근 제어 | 역할 기반 접근 통제 (RBAC) |
| 감사 추적 | 모든 접근 기록 (변경 불가) |
| 보안 취약점 | 분기별 OWASP 점검 |
