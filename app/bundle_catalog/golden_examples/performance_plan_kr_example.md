# 사업수행계획서 골든 예시 — 기획재정부 국가재정정보시스템(NAFIS) AI 고도화

## 사업 개요

**사업명**: 기획재정부 국가재정정보시스템(NAFIS) AI 고도화 사업
**계약 기간**: 2025.03.01 ~ 2026.08.31 (18개월)
**계약 금액**: 3,500,000,000원 (부가세 포함)
**발주처**: 기획재정부 재정정보화과 | **수행사**: (주)공공IT파트너스

---

## 수행 범위

**포함 (In-Scope)**:
- 국가 세출예산 집행 이상 탐지 AI 엔진 개발 (예산 전용·과목 오류·이상 집행 자동 탐지)
- 기관별 예산 집행 패턴 분석 대시보드 구축 (246개 중앙행정기관 실시간 모니터링)
- 차세대 예산회계 시스템(디브레인) 연계 API 개발 (12개 연계 테이블)
- 사용자 교육 및 운영 이관 (예산 담당자 50명, 운영 매뉴얼 3종)

**제외 (Out-of-Scope)**:
- 기존 NAFIS 핵심 예산 처리 로직 변경
- 지방재정시스템(e호조) 연계 (차기 사업)

## 품질 관리 계획

**품질 기준**: ISO/IEC 25010 소프트웨어 품질 모델 적용
- 기능 적합성: AI 이상 탐지 정확도 92% 이상 (Precision 90%, Recall 94% 이상)
- 성능 효율성: 일 집행 건수 10만 건 처리 시 응답시간 3초 이내
- 보안성: 국가정보원 보안적합성 검토 통과

**검수 절차**: 단계별 산출물 → 수행사 내부 검토 → 감독관 검토 → 협의 후 확정 → 공식 승인
