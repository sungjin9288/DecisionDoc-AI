# 프로젝트 결과보고서 골든 예시 — NIA 공공 AI 학습 데이터 구축 사업

## 경영진 요약

**프로젝트명**: NIA 공공 AI 학습 데이터 구축 사업 (2025년도)
**발주처**: 한국지능정보사회진흥원(NIA) | **계약 금액**: 2,400,000,000원
**기간**: 2025.01.15 ~ 2025.12.15 (11개월) | **팀**: 38명 (PM 1 + 데이터 팀 28 + 개발 9)

**결론(BLUF)**: 목표 데이터 50만 건 대비 52.3만 건 구축(104.6%), 납기 5일 조기 완료, 예산 집행률 97.4%로 전 지표 목표 초과 달성. AI Hub 공개 후 활용 기관 47개(목표 대비 156.7%) 확보하여 공공 AI 생태계 확장에 기여.

## 핵심 성과

**[의료 영상 판독 데이터 — 18.2만 건]**
- 국내 최대 규모 공공 의료 AI 데이터셋 구축 (흉부 X-ray 9.1만 건, MRI 5.2만 건, CT 3.9만 건)
- 판독 라벨링 품질 합격률 98.3% (방사선전문의 3인 교차 검증)
- 활용 즉시 가시 성과: 서울대병원 폐암 진단 AI 정확도 7.8%p 향상 (2025년 12월 발표)

**[법률 문서 요약·분류 데이터 — 15.7만 건]**
- 법제처 법령 해석 AI 사전 학습용 데이터 — 법령 유형 847개 분류
- AI Hub 공개 2주 만에 다운로드 기관 18개, 법학 연구자 활용 논문 3편 게재

**[행정 문서 OCR·구조화 데이터 — 18.4만 건]**
- 공문서, 민원서류, 보조금 신청서 등 행정 문서 OCR 및 필드 추출 데이터
- 정부24 민원 AI 자동처리 모델 적용 시 정확도 8.3%p 향상 (NIA 내부 검증)

## 교훈 및 개선사항

**교훈 1 — 법령 모니터링 체계 부재**
- **문제**: 의료 데이터 비식별화 기준 강화(개인정보위 고시 개정, 3월) → 재작업 6주 소요, M2 마일스톤 7일 지연
- **원인**: 사업 착수 전 관련 법령 개정 예고 모니터링 미흡
- **개선**: 향후 데이터 사업 착수 시 관련 법령 변경 예고 모니터링 체크리스트 의무 적용 (개인정보위, 복지부, 의료법 등)

**교훈 2 — 어노테이터 인력 관리**
- **문제**: 어노테이터 이직률 28% (단순 반복 작업 피로, 3~6개월차 집중 이탈)
- **원인**: 장기 단순 작업에 대한 성과 인정·보상 체계 미흡
- **개선**: 성과급 제도 도입(월 기준 초과분 추가 지급) + AI 보조 도구로 단순 반복 50% 자동화 → 어노테이터가 검수·교정에 집중하도록 역할 재편

**교훈 3 — AI Hub 공개 수요 예측**
- **문제**: AI Hub 공개 후 다운로드 기관이 목표(30개)의 156.7%(47개)로 급증 → 초기 서버 용량 부족으로 다운로드 지연 민원 발생
- **원인**: 공개 데이터 수요 예측 모델 없이 보수적 추정
- **개선**: 향후 AI Hub 공개 전 수요 예측 분석(유사 데이터셋 다운로드 추이 분석) 및 CDN 사전 확보
