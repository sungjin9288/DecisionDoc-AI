## 과업 범위

### 포함 항목 (In-Scope)
1. 계약 문서 AI 자동 검토 엔진 (불공정 조항 탐지, 법령 위반 항목 자동 플래그)
2. 계약 이행 모니터링 대시보드 (납기 준수율, 검수 현황, 하자보수 이력)
3. KONEPS(나라장터) 계약관리시스템 데이터 연계 API 개발
4. 계약 담당자 AI 어시스턴트 챗봇 (법령 Q&A, 서식 자동 작성)
5. 시스템 운영 교육 및 이관 (담당자 30명, 운영 매뉴얼 3종)

### 제외 항목 (Out-of-Scope)
- 기존 KONEPS 핵심 입찰·계약 처리 로직 변경
- 전자입찰 모듈 (별도 사업으로 발주 예정)
- 해외 조달 시스템 연계
